# Synthetic people (data)

Framtagning av testdata med fiktiva personuppgifter. Ska användas för att komma fram till vilken AI-modell som är bäst på att identifiera personuppgifter, för att i slutändan ta fram ett verktyg som maskerar dom i pdf att användas för forskning.


## Planering

#### Samla in data

### Uppgifter: 
Varje person skapar två JSON-filer: data_en.json och data_sv.json
Varje fil ska innehålla 350 objekt i samma format. Se exempel i filerna som Ernst skickade. Det ska vara ganska enkla exempel.
Man får skapa data hur man vill, men det är jätteviktigt att start- och slutindex stämmer. Man får använda AI men måste dubbelkolla att det blir rätt. 
För den engelska datan, använd en blandning av UK / USA / Kanada / Australien


Sidor man kan använda för att generera fejk-data:

Namn: https://www.behindthename.com/random/ (svenska och engelska)

Adress: https://www.fakexy.com/fake-address-generator-se (olika länder)

Personnummer: https://www.personnummer.nu/

Telefonnummer: https://www.mycountrymobile.com/sv/tools/phone-number-generator/
